# Cube Clicker

Нативна macOS гра-кликер про кубики, написана на Swift + SwiftUI + SpriteKit.

## Геймплей

- Клікай на куб → збираєш 🪵 Дерево
- Купуй будівлі → вони автоматично генерують ресурси
- Будівлі з'являються у SpriteKit-світі з анімацією

## Будівлі

| Будівля | Вартість | Ефект |
|---------|----------|-------|
| 🏚 Лісопилка | 10 🪵 | +1 🪵/сек |
| ⛏ Шахта | 20 🪵 | +1 🪨/сек |
| 🔥 Кузниця | 15 🪵 + 10 🪨 | +1 ⚙️/сек |
| 🔨 Майстерня | 30 🪵 + 5 ⚙️ | ×2 🪵 за клік |

## Вимоги

- macOS 13.0+
- Xcode 15+

## Запуск

1. Відкрий Xcode → `File → New → Project → macOS App`
2. Product Name: `CubeClicker`, Interface: SwiftUI
3. Видали автозгенеровані `ContentView.swift` і `CubeClickerApp.swift`
4. Додай всі файли з папки `CubeClicker/` до проекту
5. Додай `CubeClickerTests/GameViewModelTests.swift` до тестового таргету
6. `⌘R` — запустити, `⌘U` — тести
